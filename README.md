# 开始二周目

## 调试

`continue` 继续 断点间移动
`step in` 单步调试 逐步走 进入函数或循环
`step out` 单步跳出  跳出函数或跳出循环(函数内有断点会被卡住，非循环函数体与`continue`功能一致)
`step over` 单步跳过 逐步走 但跳过函数(函数内有断点拦截，再按则与`step in`功能一致)

字符串属性里常问到回文串，有许多种出题方式，动态规划解法放在了动态规划一章中，另外还有中心扩散法非常适合回文串的判断
